3.  Write an HTML program that includes a heading a paragraph of text, a horizontal line, and a line break.Arrange these elements to create a simple web page layout?
Ans. <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Hello HTML</title>
</head>
<body>
    <h1>HELLO HTML</h1>
    <P>THIS IS A FIRST PARAGRAPH </P>
    <HR><!--Horizontal Line--></HR>
    <p>THIS IS A SECOND PARAGRAPH</p>
    <BR><!--Break Line--></BR>
    <p>THIS IS A THIRD PARAGRAPH</p>
</body>
</html>

5. What is the DOCTYPE Declaration in HTML?
Ans. The DOCTYPE declaration specifies the type and version of HTML being used in the document.This declaration informs the web browser or parser about the rules and specifications it should follow while rendering the page.
